# USA States Guessing Game

## Overview

The USA States Guessing Game is a Python-based educational game where players guess the names of the states of the United States. Correct guesses are displayed on the map using Turtle graphics. The project uses Pandas to read state data from a CSV/Excel file and Python Turtle to plot state names on the map.

---

## Features

* Interactive USA map interface
* Guess all 50 states
* Correct answers appear on the map
* Uses CSV/Excel data handling
* Tracks score and guessed states
* Beginner-friendly Python project

---

## Technologies Used

* Python
* Pandas
* Python Turtle
* CSV/Excel file handling

---

## How It Works

1. A blank USA map is displayed using Turtle graphics.
2. The player enters a state name.
3. The program checks the CSV/Excel dataset for the state.
4. If correct, the state name is plotted on the map at its coordinates.
5. The game continues until all states are guessed or the player exits.
